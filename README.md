# 开发规范
同学们先 fork 本项目，然后 clone 主分支到本地。
1. git checkout -b feature/xxx 创建新的分支
    - xxx部分描述你要实现的功能
2. 代码写完之后，git add.
3. git commit -m "xxx"
4. git push origin feature/xxx
5. 提交 pull request

注意，pr 的描述中，请详细描述你的改动，包括改动的动机，改动的实现思路，改动的测试结果（新功能一定要写单元测试）。
